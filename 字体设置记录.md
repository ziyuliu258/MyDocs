## 设置emoji的问题
- Install **Google Noto Color Emoji**；
- Open `~/.config/fontconfig/fonts.conf`；
- Add the content
```xml
<?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "fonts.dtd">
<fontconfig>
 <alias>
  <family>emoji</family>
  <default>
   <family>Noto Color Emoji</family>
  </default>
 </alias>
 <alias>
  <family>serif</family>
  <prefer>
   <family>Noto Color Emoji</family>
  </prefer>
 </alias>
 <alias>
  <family>sans-serif</family>
  <prefer>
   <family>Noto Color Emoji</family>
  </prefer>
 </alias>
 <alias>
  <family>monospace</family>
  <prefer>
   <family>Noto Color Emoji</family>
  </prefer>
 </alias>
</fontconfig>
```
- Execute `sudo fc-cache -fv`；
- Reopen the apps.
