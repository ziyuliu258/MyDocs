## 树
### 102. 二叉树的直径
#### 关键
- 这个题目的文字表述有点让人费解，实际上意思是找到该连通图（树）中的最长路。
- 由于连通图是二叉树的形式，所以可以将这个最长路的形状分成两类：
	- 经过顶点：那就是顶点的左子树和右子树的深度的加和，这个深度是按照边数算的；
	- 不经过顶点：比如树极其不平衡，根结点没有右子树。
- 但是两种情况有共同的结构特点，就是都要经过**一个树/子树的根节点**，这个最长路一定要经过一个根节点
- 所以可以写一个求深度的函数，在这个求深度函数的基础上更新最长路`max_len`。
#### 代码实现
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        max_len = 0

        def max_depth(node: Optional[TreeNode]) -> int:  # dfs
            if node is None:
                return -1
            l_max = max_depth(node.left) + 1
            r_max = max_depth(node.right) + 1
            nonlocal max_len
            max_len = max(max_len, l_max + r_max)
            return max(l_max, r_max)
        max_depth(root)
        return max_len
```
### 108. 将有序数组转化为二叉搜索树
#### 关键
- 观察示例，可以得出最终建成的树的中序遍历的结果就是原来的数组；
- 由此可以想到，关键就是如何从一个中序遍历的序列还原成二叉树，同时还要保持平衡；
- 可以用分治（或者说二分）的方法来建树还原，这样做的好处是一定平衡，因为二分是序列/子序列的中间取根，两边的高度差肯定不会超过1，详细的证明可以看一下官方题解。
#### 代码实现
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        def div_build(left, right) -> Optional[TreeNode]:
            if left > right:
                return None
            mid = (left + right) // 2
            root = TreeNode(nums[mid])
            root.left = div_build(left, mid - 1)
            root.right = div_build(mid + 1, right)
            return root
        avltree = div_build(0, len(nums) - 1)
        return avltree
```
