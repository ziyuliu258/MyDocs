## 系列模型概述

| **模型 (年份)**        | **核心架构创新与贡献**                                             | **支持任务**               | **开发框架**              |
| ------------------ | --------------------------------------------------------- | ---------------------- | --------------------- |
| **YOLOv1** (2015)  | 首个统一的单阶段对象检测器，将边界框和类概率整合在一个网络中预测。                         | 对象检测、分类                | Darknet               |
| **YOLOv2** (2016)  | 引入多尺度训练；使用锚框维度聚类以改进先验框。                                   | 对象检测、分类                | Darknet               |
| **YOLOv3** (2018)  | 采用更深的 Darknet-53 主干网络；加入 SPP 模块和多尺度特征融合以提升小目标检测。          | 对象检测、多尺度检测             | Darknet               |
| **YOLOv4** (2020)  | 采用 Mish 激活函数和 CSPDarknet-53 主干网络以增强特征重用。                  | 对象检测、对象跟踪              | Darknet               |
| **YOLOv5** (2020)  | 提供无锚框检测头选项；使用 SiLU 激活函数和 PANet 颈部进行特征聚合。                  | 对象检测、实例分割（有限）          | PyTorch (Ultralytics) |
| **YOLOv6** (2022)  | 采用 EfficientRep 主干网络；引入无锚框对象检测模式以提高效率。                    | 对象检测、实例分割              | PyTorch               |
| **YOLOv7** (2022)  | 采用扩展的 ELAN (E-ELAN) 主干网络及模型重参数化；集成了基于 Transformer 的模块。    | 对象检测、对象跟踪、实例分割         | PyTorch               |
| **YOLOv8** (2023)  | 采用新型 C2f 主干网络和解耦头；全无锚框设计并结合生成式增强技术。                       | 对象检测、实例分割、全景分割、关键点估计   | PyTorch (Ultralytics) |
| **YOLOv9** (2024)  | 引入可编程梯度信息 (PGI)；提出 G-ELAN 架构以改进特征提取。 25                   | 对象检测、实例分割              | PyTorch               |
| **YOLOv10** (2024) | 通过一致的双重分配训练策略实现了端到端无 NMS 检测。                              | 对象检测                   | PyTorch               |
| **YOLO11** (2024)  | 加入 C3k2 CSP 瓶颈模块；引入 C2PSA（空间注意力）模块以聚焦重要区域。                | 对象检测、实例分割、姿态估计、定向检测    | PyTorch (Ultralytics) |
| **YOLOv12** (2025) | 以注意力为中心：引入高效区域注意力模块和残差 ELAN (R-ELAN) 块。                   | 对象检测                   | PyTorch               |
| **YOLOv13** (2025) | 引入超图增强自适应相关增强 (HyperACE) 和全流水线聚合分发 (FullPAD) 方案。          | 对象检测                   | PyTorch               |
| **YOLO26** (2025)  | 移除 NMS 与 DFL 实现纯净端到端推理；引入 MuSGD 优化器、ProgLoss 及 STAL 标签分配。 | 对象检测、实例分割、姿态估计、定向检测、分类 | PyTorch (Ultralytics) |
